# p123-output Manifest

This folder is organized by Portfolio123 research task. Historical docs may still mention the old flat paths; use this manifest to find the moved artifact.

## Folder Guide

| Folder | Contents |
|---|---|
| `ai-factor-validation/` | AI Factor model selection, validation, candidate strategy logs, and AI Factor campaign reports. |
| `strategy-book-api-research/` | API-estimated Strategy Book research artifacts, return panels, trial ledgers, timing overlays, and reports. |
| `strategy-book-native-validation/` | Native Portfolio123 Strategy Book inventory, validation handoff, and validation reports. |
| `ranking-api-bridge/` | API ranking-system bridge responses and native strategy details tied to that bridge. |
| `simulated-strategy-inventory/` | Simulated strategy inventory/discovery captures. |
| `factor-research/` | Standalone factor or platform research notes. |
| `misc-review/` | Holding area for artifacts that do not clearly fit a task folder. |
